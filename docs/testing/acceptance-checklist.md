# 验收清单

由产品经理和成员 C 共同维护。

## 一、语音转写

- [ ] 可以上传音频文件
- [ ] 后端接口能返回文本结果
- [ ] 返回结果包含时间戳
- [ ] 中文识别结果基本可读
- [ ] 空文件或异常文件有错误提示

## 二、纪要与关键词

- [ ] 能根据转写文本生成摘要
- [ ] 能返回关键词列表
- [ ] 能返回待办事项或结论
- [ ] 摘要内容与原文主题基本一致

## 三、搜索与回放

- [ ] 可以按关键词检索会议内容
- [ ] 搜索结果能定位到对应文本位置
- [ ] 能根据时间戳进行音频回放

## 四、前端体验

- [ ] 上传流程清晰
- [ ] 结果展示完整
- [ ] 加载状态明确
- [ ] 错误提示清楚

## 五、演示要求

- [ ] 有至少一段稳定可演示音频
- [ ] 有一套固定演示流程
- [ ] 演示时能展示转写、摘要、关键词、搜索/回放
